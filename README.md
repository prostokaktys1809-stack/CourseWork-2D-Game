2D Adventure Game (Course Work)

Опис
Курсова робота з розробки 2D пригодницької гри.
Проєкт створюється з використанням рушія Godot.
Основна ігрова логіка реалізується мовою C++ за допомогою GDExtension.

Технології
- C++
- Godot Engine
- GDExtension

Статус проєкту
Проєкт на етапі початкової розробки.
