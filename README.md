2D Adventure Game (Course Work)

2D пригодницька гра, розроблена мовою C++ з використанням рушія Godot.

 📌 Опис проєкту

Ashes of Alentania — це 2D пригодницька гра з елементами дослідження та бойової системи. 
Гравець керує персонажем, який взаємодіє зі світом, бореться з ворогами та проходить рівні.

🧠 Реалізовані можливості

- Рух персонажа
- Базова бойова система
- AI ворогів
- Система станів (FSM)
- Система керування грою

🏗 Архітектура

Проєкт реалізований з використанням принципів ООП:
- Інкапсуляція
- Наслідування
- Поліморфізм

Логіка гри реалізовуватиметься в C++, сцени та графіка — у Godot.

 🎓 Мета роботи

Розробка 2D пригодницької гри мовою C++ з використанням рушія Godot 
та закріплення навичок об’єктно-орієнтованого програмування.

Технології
- C++
- Godot Engine
- GDExtension
